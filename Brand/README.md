# Brand Documentation Overview

This directory contains all brand strategy, positioning, and guidelines for Savage by Design.

## Files in This Directory

### `brand-summary.md`
**Purpose:** Comprehensive brand documentation  
**Use for:** Pitches, repo documentation, strategic planning, onboarding

**Contains:**
- Complete brand overview and positioning statement
- Core pillars and philosophy
- Target audience definition
- Brand differentiation and competitive positioning
- Key messaging and content pillars
- Visual identity notes
- Mission, values, and future evolution

**When to use:**
- Creating new marketing materials
- Briefing partners or collaborators
- Planning content strategy
- Understanding the full brand architecture

### `voice.md`
**Purpose:** Writing guidelines and tone rules  
**Use for:** All written content, Copilot prompts

**Contains:**
- Tone and vibe guidelines
- Writing rules (sentence structure, language choices)
- Words to use and avoid
- CTA patterns
- Disclaimer tone

**When to use:**
- Writing any content (web pages, emails, social posts)
- Using GitHub Copilot to generate content
- Reviewing content for brand consistency

### `visual-direction.md`
**Purpose:** Visual design guidelines  
**Use for:** Website design, imagery, typography

**Contains:**
- Color palette direction (dark, high-contrast)
- Layout philosophy (minimalist)
- Typography guidance
- Imagery style ("rock band meets fight team")

**When to use:**
- Designing web pages or graphics
- Selecting photography
- Making typography choices

## Quick Reference

### One-Line Positioning
**Savage by Design is a performance training brand for athletes who build strength, conditioning, and resilience through deliberate systems — not chance.**

### Core Message
Elite outcomes are engineered, not accidental.

### Target Audience
Serious recreational athletes (30-50+, masters athletes) balancing high-level performance goals with real-world constraints — especially BJJ/grapplers and strength athletes.

### Brand Voice
Direct. Disciplined. Zero fluff. Hard-edged but earned. Not motivational — professional.

### Key Differentiators
- Systems over motivation
- Intent over accident
- Built for thinking athletes
- Anti-influencer, anti-hype
- Performance first, always

## Brand Guardrails

### Always
- Respect athlete intelligence
- Focus on systems and structure
- Emphasize measurable progress
- Maintain honest, direct tone
- Serve the serious recreational athlete

### Never
- Make exaggerated claims
- Use motivational/inspirational fluff
- Compromise quality for mass appeal
- Drift toward generic fitness content
- Adopt influencer culture
